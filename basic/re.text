설치 파일 설정.
npx create-react-app react-basic

1. npx : npm과 유사한 패키지 매니저
2. create-react-app : 리액트 프로젝트를 만들 때 필요한 명령어
3. react-basic : 생성할 프로젝트 폴더 이름

react-basic : 경로파일로 들어감

npm start 실행'
